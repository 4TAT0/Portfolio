WHO AM I?
웹 퍼블리셔, 웹 디자이너를 목표로 열심히 나아가는 김지나입니다.

부족하지만 실무에서 많은 것을 배우고 흡수하겠다는 생각으로 5개월의 국비교육 과정동안 HTML + CSS + J-QUERY + JAVA SCRIPT를 배우며 웹 접근성과 웹표준을 준수하여 여러 작업물들을 작업하였고, 지금도 꾸준히 JAVA SCRIPT를 공부중입니다. 아직 부족하지만 더 성장하여 디자인과 퍼블리싱 둘 다 능숙하게 해낼수 있는 다재다능한 사람이 되고 싶습니다.

두렵지 않다고는 말할 수 없습니다. 당연하게도 처음 마주하는, 한치 앞도 모르는 새로운 길로 들어섰기에 두렵기도 합니다. 그러나 동시에 하나씩 배워가며 제 것으로 만들어 가면 가슴 벅찬 기쁨과 즐거움도 느낍니다.
저는 계속 이 감정을 느끼고 싶습니다.
